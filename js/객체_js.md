# js에서의 자료형
* let --> 같은 이름을 한번만 사용 할 수 있다.
* var --> 같은 이름을 여러번 사용 가능하다.
* const --> 상수로 만들어 버린다.

# 리터널? 식별자?
    리터널 --> 변하지 않는 값,고정된 값(ex)숫자,문자열,문자 등 --> ""가 있거나 숫자로 시작한다.
    식별자 --> 함수나 변수명과 같은 것들 --> 앞이 숫자로 시작하지 않고 ""가 없다.
