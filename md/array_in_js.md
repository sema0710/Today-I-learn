# array in js
## push,pop
    push는 맨 뒤에 요소를 추가 한 뒤에 늘어난 길이를 반환한다.  
    pop은 맨 뒤의 요소를 삭제 한 뒤에 삭제한 요소를 반환한다.  
    shift는 맨 앞의 요소를 삭제 후에 삭제한 요소를 반환한다.  
    unshift는 맨 앞에 요소를 추가한 후에 늘어난 길이를 반환한다.  
> const arr = [1,2,3,4];    
arr.push(5);//5,[1,2,3,4,5]    
arr.pop();//5,[1,2,3,4]    
arr.shift();//1,[2,3,4]    
arr.unshift(0);//4,[0,2,3,4]
